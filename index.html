Ticket Finder

This is a web application developed for the University of Arizona full-stack development bootcamp. The function of thid web application is to allow users to search for events in a given city in order to view a detailed summary of each event going on in a particular city. The user can follow a ticketmaster link to purchase tickets to the event or to view further details of the event. The user is also presented with a google maps rendering which is focused on the event venue for a given event. The user can navigate aroundf the google maps embeddable on this page or they can click to travel to google maps and navigate.
